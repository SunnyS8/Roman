# 🎭 Роман — Персональный Health-Коуч на базе Betsy

**Роман** — это AI-помощник, который помогает тебе вести здоровый образ жизни. Он анализирует твоё питание, составляет тренировки, следит за здоровьем и отправляет фото, голосовые сообщения и видео-кружочки через Telegram.

## 🌟 Основные возможности

### 🍎 Питание & Калории
- **Анализ фото еды** — отправь фото блюда, Роман определит что это и посчитает калории
- **База 600+ продуктов** — каши, мясо, рыба, овощи, фрукты, молочные продукты, хлеб, напитки, соусы, закуски, азиатская кухня
- **Подсчёт БЖУ** — белки, жиры, углеводы, клетчатка
- **Дневник питания** — Роман запоминает всё что ты ел
- **Дневная сводка** — итог калорий за день в 20:00

### 💪 Тренировки
- **Составление планов** — тренировки для дома и зала
- **Рекомендации** — 3 раза в неделю (ноги+спина, грудь+руки, общий круг)
- **Отслеживание** — помня о ваших тренировках

### 🎭 Внешность & Медиа
- **Селфи** — генерирует фото Романа (используя фото как референс)
- **Голосовые сообщения** — отправляет voice notes в Telegram
- **Видео-кружочки** — создаёт видео с анимированным лицом и lip-sync к голосу
- **Печатание с точками** — три точки "..." показывают что Роман печатает ответ

### 🧠 Память & Обучение
- **Запоминает диалоги** — хранит всю историю разговора
- **Самообучение** — учится из взаимодействий с тобой
- **База знаний** — SQLite с полнотекстовым поиском
- **Расписания** — создаёт напоминания и cron-задачи

### 🏥 Здоровье
- **Диета №5** — соответствует столу №5 (для печени, поджелудочной)
- **Средиземноморские принципы** — много овощей, рыба, оливковое масло, цельнозерновые
- **Режим дня** — подъём 6:30, тренировки 20:00-21:00, отбой 22:30-23:00
- **Дробное питание** — 4-6 приёмов пищи в день (завтрак, перекусы, обед, ужин, кефир на ночь)

## 🚀 Быстрый старт

### 1. Установка

```bash
# Клонируй репо
git clone https://github.com/SunnyS8/Roman.git
cd Roman

# Установи зависимости
npm install

# Собери проект
npm run build
```

### 2. Конфигурация

Создай файл `~/.betsy/config.yaml` (на Windows это `C:\Users\YourName\.betsy\config.yaml`):

```yaml
agent:
  name: Роман
  gender: male
  avatar: /path/to/roman_avatar.png
  personality:
    tone: friendly
    style: detailed
    custom_instructions: |
      Ты — Роман, персональный health-коуч и помощник по здоровью.
      Ты заботливый, но дисциплинированный друг...

llm:
  provider: openrouter
  api_key: sk-or-v1-YOUR_OPENROUTER_KEY
  fast_model: google/gemini-2.5-flash
  strong_model: anthropic/claude-sonnet-4
  fallback_models:
    - qwen/qwen3-70b-instruct:free
    - meta-llama/llama-3.3-70b-instruct:free

telegram:
  token: YOUR_TELEGRAM_BOT_TOKEN
  owner_id: YOUR_TELEGRAM_USER_ID

channels:
  telegram:
    enabled: true
    voice:
      enabled: true
      tts_provider: minimax
      voice_id: Calm_Man
      fal_api_key: YOUR_FAL_API_KEY
    video:
      enabled: true
      fal_api_key: YOUR_FAL_API_KEY
      avatar_path: /path/to/roman_avatar.png

video:
  fal_api_key: YOUR_FAL_API_KEY
  avatar_path: /path/to/roman_avatar.png

memory:
  max_knowledge: 200
  study_interval_min: 30
  learning_enabled: true
  context_budget: 40000
```

### 3. Получи API ключи

**OpenRouter API:**
1. Зайди на https://openrouter.ai
2. Зарегистрируйся или залогинься
3. Скопируй API key из Settings
4. Добавь баланс (первые 5$ бесплатны)

**Telegram Bot Token:**
1. Напиши боту @BotFather в Telegram
2. Команда `/newbot`
3. Следуй инструкциям
4. Скопируй токен

**fal.ai API:**
1. Зайди на https://www.fal.ai
2. Зарегистрируйся
3. Перейди в Dashboard → API Keys
4. Скопируй ключ
5. **Пополни баланс** (для генерации фото, голоса, видео)

### 4. Запуск

```bash
# Для разработки
npm run dev

# Для production
npm start
```

Бот запустится на `http://localhost:3777` и подключится к Telegram.

## 📱 Как использовать в Telegram

### Команды

```
/start          — начать разговор
/help           — список команд
/status         — статус бота
/voice <текст>  — отправить голосовое сообщение
/video <текст>  — создать видео-кружочек с lip-sync
/selfie <описание> — создать селфи
/setphoto       — установить референсное фото для селфи
```

### Примеры взаимодействия

**Анализ еды:**
```
Ты: Отправляет фото курицы терияки
Роман: Анализирует фото → "Это курица терияки, примерно 150 ккал на 100г. Записал в дневник. 📝"
```

**Запрос фото:**
```
Ты: "Пришли фото"
Роман: Генерирует селфи используя свой аватар → отправляет фото
```

**Голосовое сообщение:**
```
Ты: "/voice Привет, Роман!"
Роман: Синтезирует голос → отправляет voice note
```

**Видео-кружочек:**
```
Ты: "/video Смотри, я улыбаюсь!"
Роман: Создаёт видео с анимированным лицом → отправляет video note (кружочек)
```

**Составление меню:**
```
Ты: "Давай составим меню на завтра"
Роман: Предлагает питательный завтрак, обед и ужин с учётом диеты №5
```

**Составление тренировки:**
```
Ты: "Помоги с планом тренировок"
Роман: Предлагает 3 тренировки в неделю с упражнениями для дома или зала
```

## 🎯 Основные фичи

### Food Analysis Tool (600+ продуктов)

```typescript
// Анализ продукта
await foodAnalysisTool.execute({
  action: "analyze",
  food_name: "Курица терияки",
  grams: 100
})
// Результат: { ккал: 90, белки: 18g, жиры: 1g, углеводы: 5g }

// Логирование приёма пищи
await foodAnalysisTool.execute({
  action: "log_meal",
  food_name: "Куриный суп",
  grams: 250,
  time: "13:00"
})

// Дневная сводка
await foodAnalysisTool.execute({
  action: "daily_summary"
})
```

### Typing Indicator (три точки)

Когда Роман печатает ответ, пользователь видит три точки "..." (typing indicator) которые обновляются каждые 3 секунды пока обрабатывается ответ.

### Voice Messages (голосовые сообщения)

Роман может отправлять голосовые сообщения используя MiniMax TTS через fal.ai:
- Голос: Calm_Man (спокойный мужской голос)
- Формат: OGG/MP3
- Скорость: регулируемая

### Video Notes (видео-кружочки)

Роман генерирует видео-кружочки с анимированным лицом используя SadTalker:
- Использует аватар как основу
- Синтезирует речь (MiniMax TTS)
- Анимирует лицо с lip-sync
- Отправляет как video note в Telegram

### Selfie Generation (селфи)

Роман может генерировать селфи используя Grok Image Edit:
- Использует фото как референс
- Два режима: mirror (зеркальное фото) и direct (крупное селфи)
- Автоматически определяет режим по контексту

### Диета №5

Роман автоматически следит за соответствием диете №5:
- ✅ Рыба, белое мясо, молочные продукты
- ✅ Овощи, фрукты, цельнозерновые
- ✅ Оливковое масло, небольшие порции
- ❌ Жареное, острое, очень жирное
- ❌ Алкоголь, кофе в больших количествах

## 🚨 Решение проблем

### "User is locked. Exhausted balance"
Баланс на fal.ai закончился. Решение:
1. Зайди на https://fal.ai/dashboard/billing
2. Пополни баланс
3. Перезапусти бота

### "Typing indicator не работает"
Проблема может быть в том, что Telegram уже получил ответ до того как отправился typing indicator. Это нормально для быстрых ответов (<1 сек).

### "Селфи не генерируется"
Проверь:
1. fal.ai баланс пополнен
2. API ключ fal.ai правильный в конфиге
3. Аватар существует по пути `avatar_path`
